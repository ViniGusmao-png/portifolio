Para o meu primeiro projeto eu decidi fazer um blog da franquia Resident Evil usando as minhas habildiades HTML e CSS, nesse projeto meu desafio foi deixar ele responsivo, aplicar imagens de fundo, entender como funciona as caixas texto e seus tamanhos

Esse foi um projeto proposto no curso de HTML e CSS para trabalhar com imagens de fundo usando efeito parallax e adicionando fontes internas. Eu decidi fazer uma homenagem a uma das bandas que fez parte da minha adolescencia

Nesse projeto proposto em aula, eu usei pela primeira vez o JavaScript. A ideia foi desenvolver uma página web responsiva e, com a classe Date no JS, assim que você acessar a página, ela vai pegar a hora do seu celular e exibir para você o horário, junto com uma mensagem e uma foto. Dependendo da hora que você acessar, poderá ser "Bom dia", "Boa tarde" ou "Boa noite"."

É um projeto que aborda o desafio de lidar com iframes, explorando sua funcionalidade e aprendendo a utilizá-los corretamente em diferentes situações.
